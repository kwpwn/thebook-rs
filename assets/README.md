Static assets for the book.

## Directories

| Directory | Purpose |
|---|---|
| `diagrams/` | Mermaid diagrams that are reused across chapter/appendix files |
| `screenshots/` | Sanitized screenshots from Windows VMs and tools |

---

## Screenshot rules

- Remove or blur usernames, hostnames, internal paths, tokens, keys, and proprietary telemetry.
- Record Windows build, tool version, source lab, and timestamp in the companion note.
- Prefer screenshots that prove a specific observation, not decorative images.
